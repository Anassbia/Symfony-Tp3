# TP 3 – Les formulaires en Symfony

## Rapport rapide

Dans ce TP, j’ai transformé une page HTML statique en une page Symfony dynamique en utilisant Twig et les formulaires Symfony.

### Travail réalisé

- Création d’une page produit avec Twig à partir du code HTML fourni.
- Création d’un **Form Type Symfony** pour le formulaire « Add to Cart » :
  - Champ quantité
  - Champ couleur
  - Protection CSRF activée
- Création d’un **contrôleur Symfony** pour :
  - Afficher la page produit
  - Gérer la soumission du formulaire
  - Récupérer les données envoyées par l’utilisateur
- Intégration de **Bootstrap 5.3** pour respecter le design demandé.
- Vérification du bon fonctionnement du formulaire via l’affichage des données soumises.

Le formulaire est entièrement géré par Symfony et le rendu correspond au modèle fourni dans l’énoncé.
